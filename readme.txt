== Description ==

Easily ecommerce enable any WordPress page or post. World's best PayPal integration - allows for freight calculation in PayPal checkout flow.
Manage refunds and authorisation and capture of funds.
Accept credit cards directly via PCI compliant gateway with payment page matched up seamlessly with your WordPress theme.
Requires a free [ezimerchant](http://www.ezimerchant.com) account.

== Installation ==

1. Upload wp-ezimerchant.zip to plugins directory
2. Edit files in /wp-content/plugins/wp-ezimerchant/wp-template to get product's displaying to suit

== Frequently Asked Questions ==

= How do I get paid? =

The email address registered with your WordPress blog will instantly be linked to PayPal without further interaction.
This can be changed if necessary

Credit card processing can be enabled once a merchant facility is activated with a compatible financial institution.

= What if I need help? =

Send your support questions or feedback to support@ezimerchant.com.
